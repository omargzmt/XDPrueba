XDPrueba
============

## Ejecución de proyecto

En tu máquina local clona este repositorio
```bash
git clone --recursive https://github.com/omargzmt/XDPrueba.git
cd XDPrueba
```

Para ejecutar el proyecto, escribir los siguientes comandos

```bash
docker-compose up -d
```
